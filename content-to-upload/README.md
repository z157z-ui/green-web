## ğŸ“¸ Image Guidelines

### Format
- **Photos**: JPG/JPEG (compressed for web)
- **Logos**: SVG (vector) + PNG fallback
- **Icons**: SVG preferred

### Quality
- Minimum resolution: 1200px width
- Well-lit, professional shots
- Consistent color grading
- No watermarks

### Optimization
Before uploading, compress images using:
- [TinyPNG](https://tinypng.com)
- [Squoosh](https://squoosh.app)
